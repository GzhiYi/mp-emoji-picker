# mp-emoji-picker

一个微信原生的表情选择组件。在 IOS 和 安卓中表现良好。

## 用法

直接将组件文件夹 emoji-picker 复制到项目内并进行注册。

可删除文件夹下的：

```bas
# 用于生成符合组件使用的表情数据结构
n.js

# 表情源数据
data-by-group.json
```

使用方式查看：[代码](https://github.com/GzhiYi/mp-emoji-picker/blob/master/index/index.wxml)。
